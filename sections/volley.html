HISTORIQUE DE LA SECTION DE VOLLEY-BALL

En l’an mil neuf cent quatre vingt et quinze et le 02 Mai ;s’est tenue une assemblée générale au niveau de la maison de jeunes de Seddouk de l’ASC RCSeddouk Étaient présents ce jour-là messieurs:

  • Agaoua Mhand Chérif / attaché communal des sports
  • Ichallalen Smail /président de l’ASC
  • Bouchemaa Seddik /Vice-prés //
  • Belkacemi Mohand / secrétaire adjoin //
  • Bouffala Seddik /Trésorier //
  • Tigrine Fawzi / Trésorier-adj //
  • Benslimane Nesserdine / assesseur //

Étaient présents aussi dans la salle les personnes fans de cette discipline ;

l’objet de la réunion étant l’installation d’une section de Volley-ball au sein de l’ASC . Après débat ; les membres de l’ASC RCS ,ont approuvé à l’unanimité la création de cette section . Un comité directeur est élu comme suit :

  • Mr Smaoun Salim Président
  • // Hadroug Lyazid Vice-p
  • // Belhaddad Hakim Secrétaire
  • // Meloui Karim Secr/adj
  • // Bennacer Med Trésorier
  • // Nait Bouda Hafid Tréso/adj
  • // Khiar Slimane Assesseur

En 2006 ; la section créa son propre club qui est désormais SEDDOUK VOLLEY-BALL et s’est spécialisé uniquement dans les catégories féminines ; En 2007 ; elle accéda en super division .

Faute de stade réglementaire ; toutes ses compétitions se font à l’extérieur de Seddouk .

Lieu d’entraînement : Salle de sport du TechnicumVoûte de Seddouk.
